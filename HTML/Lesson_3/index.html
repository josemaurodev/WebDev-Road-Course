Here are some useful commands


REMEMBER TO ONLY CONSIDERER THE UPPERCASE LETTERS

This is not a list, this is just a paragraph with breaking lines:

- H1, H2, H3 and so on... Is the TAG to create a title.
- P is the TAG to create a paragraph.
- BR is the TAG to break the line.
- STRONG is the TAG to create a bold word inline.
- &NBSP; is the command to create a space. Like this - -
- &LowerThan and &GreaterThan are the commands create greater and lower than symbols, like this: < or >
- © is the command to create the copyright symbol, like this: ©
