REMEMBER TO ONLY CONSIDERER THE UPPERCASE LETTERS

This is now a order list:

  1. H1, H2, H3 and so on... Is the TAG to create a title.
  2. P is the TAG to create a paragraph.
  3. STRONG is the TAG to create a bold word inline.
  4. &NBSP; is the command to create a space. Like this - -
  5. &LowerThan and &GreaterThan are the commands create greater and lower than symbols, like this: < or >
  6. &Copy is the command to create the copyright symbol, like this: © But the command should be user with uppercase.
